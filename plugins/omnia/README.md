# Omnia Code Generation

Generate and review Rust WASM crates targeting the Omnia runtime from Specify artifacts.

## Skills

| Skill | Description |
|-------|-------------|
| [crate-writer](skills/crate-writer/SKILL.md) | Generate or update Rust crates from Specify artifacts |
| [test-writer](skills/test-writer/SKILL.md) | Generate or update test suites from artifacts and crate code |
| [guest-writer](skills/guest-writer/SKILL.md) | Generate the WASM guest wrapper around domain crates |
| [code-reviewer](skills/code-reviewer/SKILL.md) | Review crates for correctness and Omnia/WASM compliance |
